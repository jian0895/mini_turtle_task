# Mini Turtle Task 🐢

Un paquete educativo de Python que simula movimientos de una tortuga mediante dibujo en texto (ASCII),
permitiendo avanzar, bajar y reiniciar la posición horizontal.  
El proyecto está diseñado para practicar **estructuración de paquetes**, **uso de variables globales**,
**imports** y **separación entre lógica e interfaz**.

---
